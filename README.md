# CLH · Prototipo web

Prototipo navegable de la Comunidad de Líderes Hispanoamericanos (CLH).

## Web pública

https://aprobarchatgpt-lang.github.io/webclh/

La web presenta la comunidad, sus cuatro pilares, el directorio de miembros, Beway, la Cumbre CLH × Beway, las propuestas para organizaciones y El Rincón de las Ideas.

## Desarrollo local

```bash
npm ci
npm run dev
```

## Publicación en GitHub Pages

Cada cambio enviado a la rama `main` ejecuta el flujo de publicación incluido en `.github/workflows/deploy-pages.yml`.
